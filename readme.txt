Use Bootstrap with the Maera theme

== Description ==

Build beautiful, bold sites that make an impression based on the Bootstrap framework!

**Powerful and simple customizer**

This shell adds the [Bootstrap](http://getbootstrap.com/) framework to your Maera theme and allows you to customize your variables using the WordPress Customizer.

**SUPPORT**

We only offer premium support on all Maera Shells. Please visit https://press.codes/support/ for more details.

== Installation ==

This plugin **requires** the Maera theme framework. You can download it from https://press.codes/
